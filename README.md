# Portfólio - Oziel do Nascimento Costa

Portfólio pessoal desenvolvido com HTML e CSS para apresentar perfil profissional, competências, projetos e certificados.

## Sobre o projeto

Este site foi criado para destacar a trajetória de Oziel do Nascimento Costa como desenvolvedor full stack em formação, com foco em JavaScript, React, Node.js, APIs e MongoDB.

O projeto principal apresentado é o **Dev Burguer**, uma aplicação web com interface de cardápio, autenticação, carrinho, pedidos e integração com backend.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- React
- Node.js
- MongoDB
